WordPress Integration Code

Copy and paste this code into your WordPress page or post:

<!-- Card Scraper Application --> <div style="margin: 20px 0;"> <iframe src="https://your-deployed-app.com" width="100%" height="800px" frameborder="0" style="border: none; border-radius: 8px; box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);" title="Card Scraper Application" allow="fullscreen"> </iframe> </div>

WordPress Plugin Integration

For better integration, you can create a WordPress shortcode:

// Add this to your theme's functions.php or a custom plugin function card_scraper_shortcode($atts) { $atts = shortcode_atts(array( 'height' => '800px', 'url' => 'https://your-deployed-app.com' ), $atts); return '<div style="margin: 20px 0;"> <iframe src="' . esc_url($atts['url']) . '" width="100%" height="' . esc_attr($atts['height']) . '" frameborder="0" style="border: none; border-radius: 8px; box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);" title="Card Scraper Application" allow="fullscreen"> </iframe> </div>'; } add_shortcode('card_scraper', 'card_scraper_shortcode');

Then use the shortcode in your posts: [card_scraper height="600px"]

Notes

  • Replace https://your-deployed-app.com with your actual deployed URL
  • Adjust the height as needed for your layout
  • Make sure your WordPress site allows iframe embedding
  • Consider adding loading states for better UX
